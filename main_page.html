My Projects

Image Generation Using GANs

Generative Adversarial Network which is popularly known as GANs is a deep learning, unsupervised machine learning technique which is proposed in year 2014.The project "Image Generation Using GANs" explores the fascinating field of Generative Adversarial Networks (GANs) and their application in creating realistic and visually compelling images. GANs are a powerful class of deep learning models that consist of two components: a generator and a discriminator.
